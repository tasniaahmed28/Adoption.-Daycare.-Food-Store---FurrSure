## FurrSure – Pet Care Management System

## Technology Stack: 
MERN (MongoDB, Express.js, React.js, Node.js)

## Description:
FurrSure is a comprehensive pet care management system designed to help pet owners, shelters, and pet businesses efficiently manage pet adoption, daycare services, and pet food supplies. 
The system supports multiple user roles with secure authentication, a responsive UI, and real-time features for seamless interaction.

This project was completed as a team project.

## Product Features
Authentication, Authorization, and User Management
Login Page – Secure user login.
Registration Page – New users can register.
OTP Authentication – Two-step verification for added security.
Role-Based Access Control (RBAC) – Admins and normal users have different privileges.
Pet Adoption Module
Add Pet for Adoption – Registered users (owners/shelters) can post pets for adoption.
Pet Listing Page – View all available pets with filters.
Pet Details Page – Shows age, breed, vaccination info, owner details, and photos.
Adoption Request Form – Interested users can send adoption requests.
Request Approval System – Owners or admins can approve or reject requests.
Adoption History – Track past adoptions per user.
Pet Daycare Module
Daycare Booking – Users can book slots for pet daycare.
Daycare Packages – Plans available for daily, weekly, and monthly packages.
Check-In / Check-Out System – Manage when pets are dropped off and picked up.
Real-Time Availability – See available spots instantly.
Pet Food Store (My Contribution)
Pet Food Display – Display available pet food products.
Pet Food Categories – Filter by type (dog, cat, bird, etc.).
Shopping Cart – Add/remove items and view total cost.
Checkout System – Secure order placement.
Pet Food Details Page – View detailed product information before purchase.
Admin Dashboard
Manage Pets – Admins can approve, edit, or remove pet listings.
Manage Products – Add, edit, or delete pet food items.
General Website Features
Dark/Light Mode – Switch between themes.
About Us Page – Explains the project’s mission.
